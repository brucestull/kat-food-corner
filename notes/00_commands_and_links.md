# Commands & Links

## Commands

- Create virtual environment and install dependencies:
    - `pipenv install django black isort flake8`

- Build in `kat-food-corner`:
    - `python setup.py sdist`
- Install in `kats`:
    - `pipenv install ..\kat-food-corner\dist\KatFoodCorner-0.1.tar.gz`
